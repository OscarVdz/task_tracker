## task_tracker
This is the website for phase 3 of our project

## Installation
1. Download all the files

2.Make sure MAMP and the sql servers are running

-go to phpmyadmin and create a new database called "task_tracker"

-click on this new database and click on the import button

-choose the sql file within this repository and then click go. This should work.
You shouldn't need to configure anything, login is all mamp default 

## Website installation
-Place the folder within your MAMP htdocs directory

-Now make sure MAMP is running and navigate to your local host
